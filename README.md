# ColorFlood

A recreation of the classic **Color Flood** puzzle game built with **C++** and **Qt**.

This project implements the core mechanics of Color Flood: starting from the top-left cell, change the color to “flood” and fill the board with a single color using the fewest moves possible.

## Features

- Desktop puzzle game implemented in Qt
- Classic Color Flood gameplay
- Customizable board size and color palette
- Simple and modular code structure


## Technologies Used

- C++
- Qt framework for GUI
- Qt Widgets for interactive gameplay

## Prerequisites

To build and run this project, you need:

- A C++ compiler
- Qt development environment installed
